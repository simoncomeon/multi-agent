# Universal Multi-Agent AI Terminal System with Project_Process Paradigm

A sophisticated multi-agent coordination system where specialized AI agents collaborate on focused **Project_Process** workflows using complete project context for intelligent AI-powered development across **any framework** (React, Vue, Python, Node.js, Java, C#, Go, Rust, etc.).

## üéØ Project_Process Paradigm

- **Single Project Focus**: All agents work on ONE Project_Process at a time
- **Complete Project Context**: AI model receives full project file contents as input
- **True AI Collaboration**: No hardcoded solutions - all generation through AI model
- **Context-Aware Intelligence**: Agents understand entire project structure
- **Collaborative Workflows**: Agents share project knowledge for better results

## ‚ú® Universal Features

- **7 Specialized Agent Roles**: Coordinator, Coder, Code Reviewer, Code Rewriter, File Manager, Git Manager, Researcher
- **Framework-Agnostic Design**: Works with React, Vue, Angular, Python, Node.js, Java, Go, C#, Rust, and more
- **Project_Process Management**: Focus on one project with complete context loading across any technology
- **Universal AI Generation**: All code/content generated through AI model with framework-aware project context
- **Standardized AI Interfaces**: Consistent input/output format for all AI operations regardless of framework
- **Smart Framework Detection**: AI automatically chooses optimal technology stack for requirements
- **Real-time Communication**: JSON-based inter-agent messaging with cross-technology project awareness
- **Cross-platform**: Native Linux/macOS support with WSL optimization
- **Automated Launchers**: Multiple startup methods for different environments and frameworks

## üöÄ Quick Start

```bash
# Start the universal AI development system
python3 smart_launcher.py ai-development
# OR
./launch_agents.sh --ai-dev

# Set Project_Process focus (in coordinator terminal)
project                    # Check current project
set_project MyApp          # Focus on specific project (any framework)
files                      # View loaded project files

# Check system status
./multi-agent status
```

### **Available Workflows:**
- `ai-development` - Universal AI-powered development (supports all frameworks)
- `code-review` - Code quality and review workflow
- `full-development` - Complete team with research capabilities
- `minimal` - Lightweight coordinator + coder setup

## ü§ñ Universal AI-Powered Project_Process Creation

The File Manager agent uses AI collaboration with project context for intelligent creation across **any framework**:

```bash
# Set project focus first
set_project MyApp

# AI-powered universal creation with framework detection
delegate 'create web app with time tracking - detect best framework (React/Vue/Angular)' to file_manager
delegate 'create Python FastAPI server with authentication and database' to file_manager  
delegate 'create Node.js Express API with TypeScript and MongoDB integration' to file_manager
delegate 'create Java Spring Boot microservice with Redis caching' to file_manager
delegate 'create Go REST API with PostgreSQL and JWT authentication' to file_manager
delegate 'enhance existing project with real-time WebSocket features' to file_manager
```

**Universal AI Collaboration Features:**
- **Framework Detection**: AI automatically chooses best technology stack
- **Complete Project Context**: AI model receives all project files as input
- **Cross-Framework Intelligence**: Works with React, Vue, Python, Node.js, Java, Go, C#, Rust, etc.
- **Context-Aware Generation**: Understands existing project structure and conventions
- **Zero Hardcoded Solutions**: Pure AI creativity with standardized interfaces

**Project Process Commands:**
- `project` - Show current project status and detected framework
- `set_project <name>` - Focus on specific project (any technology)
- `files` - View loaded project files for AI context
- `delegate "task description" to agent` - Assign framework-agnostic tasks
